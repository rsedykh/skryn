Skryn

Screenshot & Annotate menu bar app

Download for macOS

Requires macOS 14+.

Features

Menu bar app
Lives in your menu bar. Click or press a hotkey to capture.
Annotate
Arrows, lines, rectangles, blur, text, and crop before saving.
Flexible output
Copy to clipboard, save to a folder, or upload to cloud.
Edit annotations
Resize handles, drag to reposition, undo and redo.
Cloud upload
Upload existing screenshots to cloud and get a shareable link instantly.
Lightweight
Native AppKit. No Electron, no bloat, no account required.

Keyboard shortcuts

  • Arrow Drag
  • Line ⇧Drag
  • Rectangle ⌘Drag
  • Blur ⌥Drag
  • Crop ⌃Drag
  • Remove crop ⎋
  • Delete annotation ⌫
  • Undo or redo ⌘Z ⌘⇧Z
  • Text mode T
  • New line in text ⇧⏎
  • Increase or decrease text size ⌘= ⌘-
  • Save to clipboard ⌘⏎
  • Save locally ⌥⏎
  • Save to cloud ⌃⏎
  • Cancel ⌘W
  • Quit ⌘Q

Installation

  1. Download Skryn.zip from the latest release
  2. Unzip and drag Skryn.app to Applications
  3. Right-click the app and choose Open on first launch
  4. Grant Screen Recording permission when prompted
